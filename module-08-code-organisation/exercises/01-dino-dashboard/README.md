# Capstone - Dino Park Dashboard

**Mission briefing:** Build a multi-file dashboard that fetches park data, transforms it, and presents a summary. This exercise combines fetch, Promise.all, map/filter/reduce, object literals, and `this`.

## Files to implement

| File | Responsibility |
|---|---|
| `fetch-data.js` | Fetch users & posts in parallel |
| `transform.js` | Pure transform functions: `topPosters`, `postsPerUser`, `averagePostLength` |
| `dashboard.js` | `buildDashboard` — assemble the report object with a `summary()` method |

## Verify

```bash
cd starter && pnpm install && pnpm test
```

## Try it live

```bash
cd solution && pnpm start
```

Reference: `solution/` folder.
